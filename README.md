EDUCADEX - Sistema de Control Escolar

Educadex funciona como un programa administrador de elementos educativos como los alumnos, maestros, grupos y materias.

cuenta con una pantalla de inicio de sesión y dentro del sistema puedes registrar, consultar, editar y borrar información de todas esas secciones. También tiene una opción para generar reportes en PDF.


-contenido-

Lenguaje: Java 

Programa: Eclipse IDE

Base de datos: MariaDB, SQL

Diseños: Figma






1.Diseño

Utilizamos colores contraste y con un estilo puro y educativo tales como el azul marino, azul claro, dorado, negro y blanco para los textos. 
consideramos importante utilizar colores de “”soporte” para informar al usuario de ciertos eventos que están pasando en el sistema, estos son verde, rojo, y azul hover para los botones.
La tipografía es sencilla, utilizamos Times new roman para los títulos tamaño 48 y Segoe UI tamaño 20,24 para texto y botones.
Los campos de texto rellenables son sencillos pero interactivos, al ser seleccionados cambian su color a azul claro, al momento de tratar de comprobar la información estos cambian su color a verde o rojo respectivamente.
Los botones llevan una visual parecida y sencilla cambiando su color dependiendo la acción que esté determinado a realizar.
Por último los logos son variados principalmente para la selección de foto de perfil del docente.
Incluimos logos para facilitar la comprensión de las acciones a realizar dentro del programa y también para ilustrar los botones principales en la barra lateral del programa.

3.Instrucciones

Inicia sesión utilizando nombre de usuario y contraseña previamente registrados, al entrar te recibe una pantalla de inicio y puedes dirigirte a cualquiera de los 4 apartados principales, alumnos, grupos, docentes y asignaturas, en cualquier momento puedes cerrar sesión.
Los 4 apartados son muy similares ya que cuentan con las acciones de agregar, ver, editar, borrar con las que puedes manejar y realizar distintas tareas en el sistema.
